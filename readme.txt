== Description ==

Display the jobs from JobAdder to your website.

= Features =
* Display the jobs from JobAdder site to your page/post
* Filter features based on location and job type
* RSS `your-site.com/feed/?post_type=yakadanda_jobadder`

== Installation ==

1. Unzip and Upload all files to "/wp-content/plugins/".
2. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add `[yakadanda-jobadder-archive]` shortcode to your post/page.
5. Install 'JobAdder Filter' widget and 'JobAdder Details In Single' widget to sidebar.
6. Go to Yakadanda JobAdder Settings page for next instructions.

== Frequently Asked Questions ==

= How do I change default style to my preferences? =
Use /wp-content/plugins/yakadanda-jobadder/css/yakadanda-jobadder.css file as reference. Copy that file to your /wp-content/themes/active-theme/css/ folder as yakadanda-jobadder.css
